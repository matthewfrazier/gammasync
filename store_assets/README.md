# GammaSync Store Assets

This directory contains Play Store listing graphics for the GammaSync Android app.

## Required Assets

### App Icon (icon_512.png)
- **Size**: 512x512 pixels
- **Format**: PNG with no transparency
- **Style**: Clean, professional design suggesting brain/waves/therapy
- **Restrictions**: No text allowed (Google Play policy)
- **Must match**: App launcher icon style when implemented

### Feature Graphic (feature_graphic.png)
- **Size**: 1024x500 pixels
- **Format**: PNG
- **Purpose**: Clearly communicates app's therapeutic purpose
- **Content**: Should incorporate brain/neural/wave themes
