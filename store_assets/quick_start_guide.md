# Quick Start Guide for GammaSync Store Assets

## Immediate Next Steps

### Option 1: Use Canva Templates (Recommended for Speed)

1. **App Icon (512x512px)**:
   - Go to: https://www.canva.com/app-icons/templates/
   - Search: "brain health medical"
   - Select template with brain/neural theme
   - Customize colors to teal (#26A69A) and blue (#1A237E)
   - Remove all text (Google Play requirement)
   - Export as PNG at 512x512px
   - Name file: `icon_512.png`

2. **Feature Graphic (1024x500px)**:
   - Go to: https://www.canva.com/google-play-store-graphics/
   - Select "Feature Graphic" template
   - Search: "medical technology brain"
   - Add text: "GammaSync", "40Hz Gamma Brain Therapy", "Audio-Visual Entrainment"
   - Use teal-to-blue gradient background
   - Export as PNG at 1024x500px
   - Name file: `feature_graphic.png`

### Option 2: Use Figma Community Templates

1. **Search Figma Community for**:
   - "medical app icon brain therapy"
   - "health app store banner 1024x500"
   - Filter by "Open to copy"

2. **Key templates to look for**:
   - Brain/neural network themes
   - Medical/health app designs
   - Clean, professional aesthetics

### Option 3: AI Generation (Fastest)

**For App Icon**, use this prompt in Midjourney/DALL-E:
```
Minimalist app icon, 512x512px, abstract brain silhouette with gamma wave pattern, teal and blue gradient (#26A69A #1A237E), medical therapy theme, no text, clean professional design, white background --ar 1:1
```

**For Feature Graphic**, use this prompt:
```
App store feature graphic 1024x500px, "GammaSync 40Hz Gamma Brain Therapy" text, brain with neural waves, teal to blue gradient background, medical technology aesthetic, clean typography --ar 1024:500
```

## Brand Guidelines Summary

- **Primary Color**: Teal #26A69A
- **Secondary Colors**: Blue #1A237E, Purple #AB47BC
- **Style**: Clean, professional, medical/scientific
- **Theme**: Brain entrainment, 40Hz gamma waves, therapy
- **Target**: Adults interested in brain health/optimization

## File Requirements Checklist

### App Icon (icon_512.png)
- [ ] Exactly 512x512 pixels
- [ ] PNG format, no transparency
- [ ] No text anywhere on icon
- [ ] Uses teal/blue color scheme
- [ ] Brain/wave/neural theme
- [ ] Readable at small sizes (test at 48px)

### Feature Graphic (feature_graphic.png)
- [ ] Exactly 1024x500 pixels
- [ ] PNG format
- [ ] Includes "GammaSync" branding
- [ ] Communicates 40Hz brain therapy purpose
- [ ] High contrast text
- [ ] Professional medical aesthetic

## Upload Instructions

1. Save both files to `/store_assets/` directory
2. Verify file names: `icon_512.png` and `feature_graphic.png`
3. Test icon visibility at multiple sizes
4. Review feature graphic text readability
5. Ready for Play Store upload
