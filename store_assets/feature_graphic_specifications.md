# GammaSync Feature Graphic Specifications (1024x500px)

## Design Requirements
- **Dimensions**: 1024x500 pixels exactly
- **Format**: PNG
- **Purpose**: Primary visual for Play Store listing - must clearly communicate app function
- **Safe zones**: Keep critical text/imagery within 924x400px center area to avoid cropping

## Visual Concept
Communicate "Brain entrainment therapy using 40Hz gamma waves"

### Layout Structure (1024x500px)
```
[      Background Brain/Wave Visualization       ]
[  Logo  |  "40Hz Gamma Brain"   |   Visual Demo ]
[  Area  |  "Entrainment Therapy" | (Waveform)  ]
```

### Key Visual Elements:

1. **Background**: 
   - Subtle neural network pattern or brain connectivity map
   - Gradient from teal to deep blue/purple
   - Semi-transparent to allow text readability

2. **Left section (300px)**:
   - GammaSync app icon/logo
   - App name "GammaSync" in clean typography

3. **Center section (400px)**:
   - Main headline: "40Hz Gamma Brain Therapy"
   - Subtitle: "Audio-Visual Entrainment"
   - Clean, medical/scientific typography

4. **Right section (324px)**:
   - Visual representation of 40Hz wave
   - Brain silhouette with wave emanation
   - XREAL glasses integration hint

## Typography
- **Primary font**: Inter or Roboto (clean, scientific)
- **Headline**: 36-42px, bold weight
- **Subtitle**: 24-28px, medium weight
- **Color**: White or very light text (#FFFFFF or #F8F9FA)
- **Ensure**: High contrast against background
